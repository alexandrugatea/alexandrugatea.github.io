Demonstration of Bubbling Events

When you click inside one of several nested elements (like boxes within boxes), you trigger a "click" event on the specific element you clicked. This event then bubbles up to each parent element in turn, all the way up the hierarchy until it can't go any higher.

Demonstration of Capturing Events

Event capturing is essentially the flip side of bubbling. Instead of starting at the clicked element and moving outward, capturing begins at the top of the hierarchy and works its way down to the target element.

Demonstration of stopping bubbling/capturing

If you need to prevent an event from bubbling up or capturing down, you can use the "stopPropagation" method within an event handler. This stops the event in its tracks, meaning it won't continue to propagate through other elements.
